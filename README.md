# 📋 CRUD with JavaScript

A lightweight, client-side **CRUD application** built with vanilla JavaScript. Create, read, update, and delete records directly in the browser — no frameworks, no backend, no dependencies.

## ✨ Features

- ➕ **Create** — Add new records via a simple form
- 📖 **Read** — View all saved records in a clean list/table
- ✏️ **Update** — Edit any existing record in place
- 🗑️ **Delete** — Remove records instantly
- 💾 **LocalStorage** — Data persists after page refresh *(if implemented)*
- 📱 **Responsive** — Works on desktop and mobile

## 🏃 Getting Started

No installation needed. Just clone and open in your browser.

```bash
# Clone the repository
git clone https://github.com/AdriHamzallari/CRUD-with-JS.git

# Navigate into the project
cd CRUD-with-JS

# Open in browser
open index.html
```

Or simply [download the ZIP](https://github.com/AdriHamzallari/CRUD-with-JS/archive/refs/heads/main.zip) and open `index.html`.
